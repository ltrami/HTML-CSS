Capítulo 3: Formatações

Em HTML5 temos vários tipos de formatação de texto.

Negrito / Destaque

Textos em negrito podem ser inseridos utilizando uma das seguintes tags: <b> e <strong>. No entanto, a tag <b> é uma tag não-semântica (dá forma ao invés de contexto), e por conta disso, em HTML5 devemos utilizar a tag <strong> em seu lugar, por se tratar de uma tag semântica. Em HTML5 temos a tag <strong> para dar destaque ao texto, substituindo assim o uso da tag <b>.

Itálico / Ênfase

Podemos também inserir textos em itálico utilizando duas tags: <i> e <em>. Porém, assim como acontece com textos em negrito, em HTML5 prefere-se o uso da tag <em> para dar destaque ao texto (forma semântica).

Texto Marcado

Para inserir um texto marcado, como podemos fazer em um caderno ou livro usando um marca-texto, utilizamos a tag <mark>. Por padrão, o texto fica marcado na cor amarela, mas é possível alterar a cor da marcação utilizando CSS (usando o seletor <style>, mas isso veremos mais pra frente.

Alterar tamanho do texto

É possível utilizar a tag <small> para diminuir o tamanho do texto, embora seja preferível alterar tamanho de texto utilizando CSS. A tag <big> está obsoleta e não deve ser usada, sendo mencionada aqui apenas para servir de lembrete do que NÂO USAR.

Texto Deletado (strikethrough text)

Quando queremos indicar que um trecho do texto deve ser lido, mas desconsiderado, utilizamos a tag <del>.

Texto Inserido

A utilização de textos inseridos serve tanto para dar ênfase ao trecho como também para indicar que aquele trecho/texto foi inserido depois. Visualmente, o efeito é o mesmo de um texto sublinhado, porém, a tag que era utilizada para sublinhar textos <u> está obsoleta, e em HTML5, utilizamos em seu lugar a tag semântica <ins>.

Texto sobrescrito e subescrito

Exemplos de texto sobrescrito são representações de potência em cálculos matemáticos, como X elevado à 2. Para inserir este tipo de texto corretamente em HTML5, utilizamos a tag <sup>imediatamente antes do caractere que queremos alterar. O resultado é este aqui: X2.

Já os textos subescritos, como por exemplo, a representação da fórmula do oxigênio (O2), são inseridos utilizando a tag <sub> imediatamente antes do caractere que será alterado.

Citações Simples

Para inserir citações simples, usamos a tag <q>, que indica quote.

Abreviações

Quando usamos abreviações, como por exemplo, HTML, devemos utilizar a tag <abbr title="Escreva aqui o que a abreviação significa"> para ajudar mecânismos de indexação e leitores de tela.

Código Fonte / Pré-Formatação

Quando queremos inserir um código em linguagem de programação, usamos a tag <code> para facilitar a leitura, convertendo o texto envelpado pela tag em fonte monospace. Porém, para manter a integridade visual (identação) do código, é necessário envelopar a tag <code> com a tag <pre>, que serve para manter a pré-formatação do texto.

Exemplo: O comando document.getElementById é usado em linguagem JavaScript.

           
            num = int(input('Digite um número'))
            if num % 2 ** 0:
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num} é ÍMPAR')
            print('fim do programa')

Note que a pré-formatação considera a posição do texto no editor (por isso o código acima aparece mais para o meio da tela). Podemos alterar isso ao "puxar" o código mais para a esquerda no código fonte da página, como podemos ver abaixo (no exemplo a seguir, o texto pré-formatado "toca" a "parede" esquerda da tela do editor, no código fonte da página):

        
num = int(input('Digite um número'))
if num % 2 ** 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('fim do programa')

Isto é chamado de Tabulação, e é feito no editor selecionando o texto desejado, e em seguida, segurando a tecla SHIFT e apertando a tecla TAB.
